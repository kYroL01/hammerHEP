### Usage of ./hammerHEP:
  -addr string
        Address to send packets (default "localhost")
  -port string
        Port to send packets (default "9060")
  -prot string
        Supported protocols are hep,ipfix (default "hep")
  -rate int
        How many packets per second to send (default 1)
